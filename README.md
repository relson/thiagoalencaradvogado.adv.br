# Thiago Alencar Advogado

Site do escritório Alencar Advogados — [thiagoalencaradvogado.adv.br](https://thiagoalencaradvogado.adv.br)

## Desenvolvimento local

```bash
cd docs
bundle install
bundle exec jekyll serve
```

Acesse em http://localhost:4000

## Deploy

O site é publicado via GitHub Pages a partir da pasta `docs/`.
